Problem Statment -> create a tcp socket server which connects
one client at a time. client sends a message to server and 
after recieving this server replies back to client with uppercase
letter of message.

Compile:
gcc server.c -o server
gcc client.c -o client

RUN:( execute each command in a new terminal )
./server
./client
...
